## Dados Brutos (Raw Data)

Esse projeto utiliza dados reais de operação de controle de itens vencidos em estabelecimentos comerciais, no entanto, por questões de privacidade, segurança e governança de dados.

Antes de serem utilizados, foi realizado um processo de tratamento que inclui:

* **Anonimização de informações sensíveis**
    
    Identificadores como códigos de lojas e bandeiras como referências diretas aos estabelecimentos foram transformados para impedir identificação.

* **Remoção de dados descritivos**

    As informações de descrição de produtos e códigos PLU foram completamente removidas, evitando exposição de fornecedores e dados comerciais. 

* **Padronização do documento**

    Originalmente o arquivo se encontrava como uma planilha excel com diferentes informações anexadas em abas, foi extraído somente os dados principais e transformado em csv para maior compatibilidade.

> *Os dados disponibilizados no repositório representam uma versão polída e segura da base original, não permitindo identificação de indivíduos, empresas, lojas ou produtos.*
